# 🍎 Sistema de Inventario de Alimentos

Un sistema moderno y responsive para la gestión de inventario de alimentos, construido con React y Vite. Perfecto para pequeños negocios, restaurantes o uso personal.

## ✨ Características

- 📊 **Dashboard con resumen estadístico** - Total de productos, unidades, stock bajo y productos agotados
- ➕ **Agregar nuevos productos** - Formulario intuitivo con categorías predefinidas
- ✏️ **Editar cantidades** - Actualiza el stock fácilmente
- 🗑️ **Eliminar productos** - Remueve items del inventario
- 🔍 **Búsqueda en tiempo real** - Filtra por nombre o categoría
- 🎨 **Indicadores visuales de stock**:
  - 🟢 Verde: Stock normal
  - 🟡 Amarillo: Stock bajo (≤ 5 unidades)
  - 🔴 Rojo: Agotado (0 unidades)
- 📱 **Diseño responsive** - Funciona perfectamente en móviles, tablets y escritorio
- ⚡ **Animaciones suaves** - Experiencia de usuario fluida

## 🛠️ Tecnologías Utilizadas

- **React 18** - Biblioteca para construir interfaces de usuario
- **Vite** - Herramienta de construcción rápida
- **CSS3** - Estilos modernos con Flexbox y Grid
- **Hooks de React** - useState, useEffect para manejo de estado

## 📋 Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

- **Node.js** (versión 14 o superior)
- **npm** (normalmente viene con Node.js) o **yarn**

Puedes verificar las versiones instaladas con:
```bash
node --version
npm --version
```

## 🚀 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/leaox77/sistema-inventario-alimentos.git
cd sistema-inventario-alimentos
```

2. Instala las dependencias:
```bash
npm install
# o
yarn install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
# o
yarn dev
```
4. Abre tu navegador y visita `http://localhost:5173` para ver la aplicación en acción.
